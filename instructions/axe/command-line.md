# a11y testing with pa11y
Requirements : Node 8

## Setting up
 ```
 npm install axe-cli
  ```
  add a script in package.json for the commandline tool:
 ```
   "scripts": {
    "axe-command": "axe [https://path/to/page]"
  }
 ```
## 1. A basic run

 ```
 npm run axe-command
  ```
